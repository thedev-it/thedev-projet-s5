# thedev-projet-s5



Groupe Name :  The dev 

Projet 1 Topic : Site vitrine associatif /ONG 

Description de projest : 
 
 Ce projet consiste à concevoir et développer un site vitrine pour une association ou une ONG fictive.
En tant qu'agence, l'objectif principal de votre groupe est de créer une plateforme web qui informe clairement les visiteurs sur les actions menées par l'organisation et qui donne envie de soutenir sa cause. 
Le projet impose de livrer les quatre pages clés suivantes :

Accueil : Présentation globale de la mission de l'association et mise en avant de chiffres clés.

Nos actions / missions : Une section présentant la liste ou une grille des différents projets menés par l'organisation.

Événements à venir : Un agenda ou une liste chronologique des dates et événements importants à venir.

Contact / Faire un don : Une page contenant un formulaire statique
